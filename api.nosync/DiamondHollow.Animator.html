Constructors

Animator(DiamondHollowGame, Level, String, Int32, Nullable<Rectangle>)

An animation object can have multiple states, each with unique animation. States are keyed by strings, the default key is "default". Animating can be paused/resumed and a callback can be set to run when an animation loop completes.
